Documentation de Umaservices2

Présentation

Umaservices2 est un framework de développement réalisé par Umazuma.
Il est basé sur des bibliothèques (ou framework) existantes et largement partagées.

Ces bibliothèques sont :

  • Silex (Symfony) pour l'applicatif serveur
  • JQuery et JSviews pour l'applicatif client
  • Bootstrap et HTML5Boilerplate pour la présentation
  • Modernizr et Webshims pour la compatibilité internavigateur

 

Objectifs

Umaservices2 a pour objectif de proposer une gestion client / serveur complètement réorientée vers la réduction des échanges tant en quantité qu'en format pour permettre un allègement des charges de serveur et une gestion modulable coté client.

Les principes de base sont donc :

  • Le navigteur demande un service (un menu, une page de détail, un formulaire, etc)
  • Un script intercepte cette demande en local (JQuery) et appelle une combinaison de template et de données
  • Le serveur (ou le serveur CDN) délivre des templates Javascript exploitable par JSViews
  • Le serveur délivre des données au format JSON
  • L'ensemble template + données JSON est traité localement par le navigateur (JQuery)
