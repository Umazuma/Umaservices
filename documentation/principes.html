Principes de fonctionnement

La page web index.html est structurée à partir de HTML5Boilerplate

<!doctype html>
<html class="no-js">
<head>
<meta charset="utf-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<title>Umamodel</title>
<meta name="description" content="">
<meta name="viewport" content="width=device-width, initial-scale=1">
<link rel="stylesheet" href="css/normalize.css">
<link rel="stylesheet" href="css/main.css">
<script src="js/vendor/modernizr-2.7.1.min.js"></script> 
</head>
<body>
<div id="container"></div>
<script src="js/main.js"></script>
</body>
</html>

Le script Modernizr va s'occuper de charger les scripts et les feuilles de styles complémentaires. cela se passe dans le script main.js
